JSF Interceptor example (Jerry)

In this example

ComponentInitializer (be.atbash.ee.jsf.jerry.example.interceptor.DummyInitializer) which
  • Adds the text Hijacked to all OutputLabel
  • Adds the text Interceptor :) as placeholder in InputText
  • Changes the background colour of required InputText components.

Custom configuration (be.atbash.ee.jsf.jerry.example.jerry.CustomConfigurator) which actually doesn't do anything special

StartupEvent capture to perform some initialization when war is deployed.
